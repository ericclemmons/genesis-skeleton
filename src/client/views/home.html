Genesis Skeleton

Zero to Live in minutes.

About

Genesis Skeleton aims to be the best starting point for your next Express + AngularJS app. Assets are automatically concatenated & minified with Grunt, ready to change & deploy.

Get Started in 5 Easy Steps

  1. Download or git clone git://github.com/ericclemmons/genesis-skeleton.git
  2. Install dependencies: npm install
  3. Preview your changes live: grunt server
  4. Create a Heroku server: heroku create
  5. Launch your app! git push heroku master

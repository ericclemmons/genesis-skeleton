Get Started in 5 Easy Steps

  1. Download or git clone
  2. Install dependencies: npm install
  3. Preview your changes live: grunt server
  4. Create a Heroku server: heroku create
  5. Launch your app! git push heroku master
You should already have NodeJS installed, as well as Grunt and Bower globally: 
npm install -g grunt-cli bower

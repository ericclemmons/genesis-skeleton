Features

 Structured

Based on both client & server-side convention, every part of your application has a place, no matter how big it gets.

 Automated build process

Leveraging Grunt, everything from linting your scripts to pre-processing your CSS and images is done for you.

 WYSIWYG

Simply run grunt server and watch your changes instantly via LiveReload.

Optimized for Angular

Angular templates are automatically pre-loaded so your app loads instantly without any unecessary requests.

 Optimized

From CSS pre-processing to image optimization, everything is ready for production with a simple grunt build.

 Heroku-ready

You're only a git push heroku master away from showing off your latest project.

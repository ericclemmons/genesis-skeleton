Frequently Asked Questions

Why AngularJS instead of a different framework?

For robust applications with a lot of moving parts, AngularJS has shown to support both rapid prototypes and complex applications requiring well-structured architecture.

Don't fret, though! You're only a simple bower install away from having your favorite framework.

How is this different from Yeoman and other generators?

Yeoman is a fantastic generator project. This project is, however, *not* a generator. Genesis Skeleton is an opinionated starter application meant for you to fork, modify, and start each project with a well-organized and flexible foundation.
